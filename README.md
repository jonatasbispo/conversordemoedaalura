# 💱 Conversor de Moedas em Java

Este é um projeto simples de **conversor de moedas**, desenvolvido em Java, que consome uma API externa para realizar conversões com **taxas de câmbio em tempo real**. O projeto oferece interação via console e demonstra a integração com web services, manipulação de JSON e boas práticas de programação.

---

## 📌 Funcionalidades

- Conversão entre moedas com dados atualizados em tempo real.
- Consumo da API ExchangeRate-API.
- Interface de texto simples via console.
- Base para expandir com mais funcionalidades no futuro (como menu interativo, múltiplas conversões etc).

---

## 🛠️ Tecnologias utilizadas

- Java 17
- IntelliJ IDEA Community Edition
- Biblioteca `org.json` para manipulação de JSON
- API pública: [ExchangeRate-API](https://www.exchangerate-api.com)
